Sondata Studios

"taking inspiration from sounds"

Sondata represents:

a collective of craft practitioners from across Scotland.

We originally formed as a collective of textile practitioners, so hand weavers, knitters and screen printers and have recently expanded it to include other making disciplines. So ceramics, jewelry maker and furniture design.

Our vision:

To create handmade products that are all underpinned by taking inspiration from sounds.

On this page we have created our own pattern playground for you to upload audio files and interact with the visualised outputted audio. These can be edited and downloaded at your leisure.

Help Section

A quick guide on how to use the Spectogram:

1. find an audio file

You will need a local audio file to upload

You can upload an audio file by clicking the browse button. If you dont have one - press play for a sample bird noise.

2. Edit the Audio

Hover over an option for additional helps

Once you are ready you can change the audio settings. Press "Start Audio" to see the changes in the audio output.

3. Frame the sound file  

Zoom  in/out   

Once the audio has started feel free to frame the audio in anyway you want by using the zoom functions 

4. Download

Download or save

Once you are happy with the output. Press the download pattern button for a PNG or press the Submit Image button to save
